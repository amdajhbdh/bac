# Hydrocarbons

## Definition
Organic compounds containing only carbon and hydrogen atoms.

## Classification

### 1. Alkanes (Saturated)
Single bonds only (C-C)

**General Formula**: C_nH₂_n₊₂

**Examples**:
- Methane: CH₄
- Ethane: C_2H₆
- Propane: C_3H₈
- Butane: C₄H₁₀

**Properties**:
- Least reactive
- Undergo substitution reactions
- Combustion: C_nH₂_n₊₂ + O₂ → CO₂ + H₂O

### 2. Alkenes (Unsaturated)
One or more double bonds (C=C)

**General Formula**: C_nH₂_n

**Examples**:
- Ethene: C_2H₄
- Propene: C_3H₆
- Butene: C₄H₈

**Properties**:
- More reactive than alkanes
- Undergo addition reactions
- Test: Decolorize bromine water

### 3. Alkynes (Unsaturated)
One or more triple bonds (C≡C)

**General Formula**: C_nH₂_n₋₂

**Examples**:
- Ethyne (Acetylene): C_2H₂
- Propyne: C_3H₄

**Properties**:
- Most reactive
- Undergo addition reactions
- Used in welding torches

### 4. Aromatic Hydrocarbons
Contain benzene ring (C₆H₆)

**Structure**: Hexagonal ring with alternating double bonds

**Examples**:
- Benzene: C₆H₆
- Toluene: C₇H₈
- Naphthalene: C_1₀H₈

## Isomers
Compounds with same molecular formula but different structures

**Example**: C₄H₁₀
- n-butane (straight chain)
- isobutane (branched)

## Reactions

### Combustion
Complete: C_nH_m + O₂ → CO₂ + H₂O
Incomplete: C_nH_m + O₂ → CO + H₂O (or C + H₂O)

### Substitution (Alkanes)
CH₄ + Cl₂ → CH₃Cl + HCl (with UV light)

### Addition (Alkenes)
C_2H₄ + H₂ → C_2H₆ (hydrogenation)
C_2H₄ + Br₂ → C_2H₄Br₂
